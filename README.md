## Choice of Technology
### FrontEnd
HTML, CSS, JavaScript
jQuery
Reason: jQuery has a wealth of plug-ins and extensions, which greatly simplifies the construction of the HTML difficulty, but also solves the problem of browser incompatibility.
### Databse
MongoDB
Reason: Because it is flexible, allowing us to easily adapt our data structure as needed. MongoDB Atlas helps our team work together and manage the database in one place. Plus, many team members are already familiar with MongoDB, which makes it easier for us to use it effectively.
### BackEnd
Node.js, Express.js, socket.io
